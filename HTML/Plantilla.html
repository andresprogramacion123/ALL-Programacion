Plantilla de html.

Hola a partir de ahora vamos a comenzar a documentar como escribir y realizar codigo en html. Vamos a observar algunas etiquetas y a relizar una plantilla generica.
HTML es un lenguaje de estructura que va de la mano con CSS y Javascript.
La manera de realizar un codigo en HTML es a partir de etiquetas las cuales se encuentran en el interior de los simbolos menor que y mayor que. La gran mayoria de etiquetas se deben cerrar y abrir. La manera de abrirlas es poniendo su nombre en el interior de los dos simbolos y la manera de cerrarla es ubicar el nombre de la etiqueta donde lo antecede el simbolo / en el interior de las desigualdades.

Ahora veamos como es que se escribe un documento HTML.
Siempre a la hora de comenzar a realizar un codigo en HTML se debe comenzar con la etiqueta html, la anterior etiqueta no debe faltar y debe cerrarse al final del documento.
Seguido de la etiqueta html continuamos con la etiqueta head tambien llamada cabecera. Esta etiqueta contiene informacion que el usuario no puede ver y que se hace util para el navegador. De nuevo esta etiqueta hay que cerrarla.
Segudidamente esta la etiqueta body o cuerpo esta es la etiqueta donde se va comenzar a escribir todo lo que va contener la pagina web y que corresponde a lo que va observar el usuario.
De esta manera tenemos lo mas fundamental para comenzar a realizar un codigo en html, por el momento las etiquetas html, head y body son las etiquetas fundamentales y que no deben faltar a la hora de realizar un script en html.

Ahora veamos el uso de mas etiquetas y sus atributos. Vamos a comenzar con la etiquea h1. La etiqueta h1 es una etiqueta utilizada para poner titulos en la pagina web. Se debe tener en cuenta que por pagina web solo debe de haber un titulo, es decir, la etiqueta h1 solo debe aparecer una vez en codigo.

Ahora veamos el uso de enlaces

Los enlaces son aquellos textos subrayados los cuales al darle click nos envia hacia otro lugar. La forma de generar un enlace es con la etiqueta a donde esta etiqueta en su interior posee un atributo llamado href el cual corresponde al enlace (URL) o lugar al que queremos que nos envie el texto. Para el caso de una URL es necesario agregar el protocolo https. Debemos de tener en cuenta que lo anterior es con una ruta absoluta.
Hay dos tipos de rutas: ruta absoluta y ruta relativa.
La ruta relativa hace referencia a un archivo que se encuentra en la misma carpeta o en una caperta inferior a la ruta donde esta el archivo HTML.
Ahora al hacer click en el enlace lo que hace es que nos envia a este lugar pero nos desaparece nuestro sitio web. Para seguir conservando nuestro sitio web y que se abra el enlace en otra pestaña lo que hacemos es poner un atributo en la etiqueta a el cual se llama target="_blank" de manera que al hacer click nos conserva el sitio web y abre en una nueva pestaña el enlace.

Ahora veamos el tema de listas

Hay dos tipos de listas: listas ordenadas y listas desordenadas.
Listas desordenadas: se agregan con la etiqueta ul y en su interior va la etiqueta li, luego en la etiqueta li podemos aplicar la etiqueta a que es para enlaces para convertir este item en enlace.
Vale aclarar que dentro de una lista podemos tener mas listas.
Estas listas desordenadas se suelen utilizar en las paginas web para hacer menu de navegacion o para mostrar contenido.
Ahora veamos las listas ordenadas.
La lista ordenada: se forma con la etiqueta ol, y de nuevo va en su interior la etiqueta li donde a su vez podemos tener mas listas en cada item.
Las listas ordenadas no se suelen utilizar mucho aunque pueden ser utilizadas para representar datos.

Veamos el tema de imagenes

La etiqueta para agregar imagenes es img la cual posee varios atributos y se cierra por si sola es decir al finalizar se escribe el simbolo /.
Se le debe indicar la ruta de la imagen por medio del atributo src, en caso de que se quiera cambiar el tamaño de la imagen agregamos el atributo width y height. Otro atributo muy importante es el atributo alt el cual consiste en poner el nombre de la imagen y que saldra en caso de que la imagen no cargue o de que su ruta este equivocada y por ultimo otro atributo es title el cual consiste en ponerle titulo a la imagen que solo sale cuando el click esta encima de la imagen

Veamos el tema de los formularios

Un formulario es una parte de nuestro sitio web donde el usuario puede interactuar ingresando texto o datos y la pagina los procesa y luego envia una respuesta.
La etiqueta de formulario es la etiqueta form la cual puede rebir varios atributos.
Uno de los atributos es action="URL" el cual recibe como valor una URL que se encarga de procesar los datos del formulario y el otro atributo es method="POST" o method="GET" el cual veremos mas adelante.
Dentro de la etiqueta form se encuentras varias etiquetas:
la etiqueta input la cual es la encargada de recibir el valor a ingresar. Tiene varios atributos entre ellos tenemos type="" el cual corresponde a los valores que va recibir si es el caso de texto entonces type="texto", otro atributo es value="" el cual me dice que debe de ingresar el usuario el problema es que toca borrar el mensaje. Por esta razon el atributo placeholder="Escribe tu nombre" es mas utilizado pues no es necesario borrar en la caja de texto.
Otro atributo es size="" el cual es el tamaño de la caja de texto, otro atributo es maxlength="23" el cual corresponde al numero maximo de caracteres que se permite. otro atributo es name="ussuario" lo cual es importante a la hora de enviar los datos, el id es otro atributo para asignar a otras etiquetas. Si agregamos atributo type="password" permite ingresar contraseñas pero sin poderse ver.
type="email" sirve para corraborar que se ingresa un correo.
Otro atributo type es type="checkbox" o type="radio", tambien esta type=button" y value="enviar" que no realiza ninguna accion. Para un buton que realice una opcion el atributo es type="submit" y permite enviar los datos.
Otra etiqueta que se tiene es select y en cual podemos poner otra etiqueta que permita escoger opciones con la etiqueta option
